Research

Cloud Computing

My research on Cloud Computing focus on resource management in the form of Virtual Machine.
Taking VM as a complicated unit, with individual CPU, Memory, disk, and network, I use two kinds of method to manage them.

    Modified Linear Programming
  • 1. Abstract Cost, Performance, and Availability using VM;
  • 2. Based on SLA, assign different priorities to these three aspects;
  • 3. Linear programming of first two aspects, and get a reasonable range of output;
  • 4. Add the third aspects, select the best result within the previously range.
    Dynamic Programming
  • 1. Abstract Cost and Performance, and Availability using VM;
  • 2. Calculate the values of all these three aspects by adding each kinds of VM from 0 to n;
  • 3. Backwardly find the best solution based on the 2D table generated by step 2.

Mobile Privacy

My research on Mobile users' privacy is based on the Data Over-Collection, I presented at 05/2016. Data over-collection means smartphones apps collect users' data more than its original function while within the permission scope.
My research work include analyzing the motivation, formalizing the relation between privacy and data, and proposing a solution.
